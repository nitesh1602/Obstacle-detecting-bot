# Obstacle Detecting Bot
 A bot which detects the obstacle in its path by using ultrosonic sensor and then decides the next path.
You need AFMotor dependency NewPing dependency and Servo dependency.
Hardware: 
     Arduino
     L-Shaped Motors
     Motor Driver Shield
     Ultrasonic Sensor
     Servo Motors
     Wheels
     Jumper wires
     Chasis
     Battery
